
# Project


**Le but de ce projet est de réaliser une API qui assurera les éléments suivants:**
- Créer une documentation API depuis la description des endpoints format json
- Génération du code OpenAI associé
- Génération de la documentation API en format Html
- Capacité à importer une documentation au format OpenAI

Ce projet est lié au site internet Erdline, qui a pour but de permettre de créer une    
documentation API de façon graphique. Et non en apprenant à coder avec le langage   
OpenAI.     

Le site Erdline est en cours de construction.
